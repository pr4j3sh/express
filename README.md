# Express

This is an express js template, pre-integrated with utilities, so that you can focus on creating API(s).

## Usage

- Clone using `@pr4j3sh/frames`

```bash
npm create @pr4j3sh/frames@latest express myapi
```

- Run using `dev`

```bash
npm run dev
```
